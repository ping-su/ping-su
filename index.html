About

我是 Ping，一位注重使用者體驗的 UI/UX 設計師，累積超過 10 年的實務經驗。
專長於 Web/App 架構規劃、介面設計與優化，同時也熟悉 前端 HTML/CSS ，可與工程團隊無縫協作。
我擅長從使用者行為與商業需求中找出關鍵問題，整合資訊架構與設計系統，協助團隊打造符合目標的數位產品。
喜歡研究技術與設計趨勢，也樂於持續學習與優化流程，讓設計真正解決問題，而不只是做得漂亮。

我擅長於 🧠

  • 使用者體驗規劃：行為分析、動線設計、資訊架構整理
  • 介面設計與設計系統：視覺風格統一、元件系統化、互動設計
  • 原型與驗證：高保真原型、用戶測試、快速迭代
  • 前端支援：HTML/CSS 標準語法、與工程師合作串接落地

熟悉工具 🛠️

  • Figma
  • Zeplin
  • Visual Studio Code
  • GitHub

設計流程 🔄

  • 需求理解

    －與案主、PM 深入討論問題與目標，釐清需求與限制。

  • 功能規劃與原型設計

    －建立功能架構、內容流程，產出高保真原型與使用路徑。

  • 視覺設計與開發協作

    －建立 Design System、設計 UI 元件，與工程師協作技術落地。

  • 測試與優化

    －根據回饋進行設計調整，優化用戶體驗與操作流暢度。
